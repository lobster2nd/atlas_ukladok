## Установка проекта локально (Linux)  
+ Склонировать репозиторий и перейти в него в командной строке:  
```
git clone https://github.com/lobster2nd/atlas_ukladok .git  
cd atlas_ukladok  
```  

+ Запуск сервера:  

```
make up
```

+ Остановить сервер:  

```
make down
```

+ Документация
```
http://0.0.0.0:8000/swagger/
```
